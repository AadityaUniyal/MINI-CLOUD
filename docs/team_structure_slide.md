# 👥 MiniCloud Team Roles & Tech Stack

## 👤 Member 1: Database & Core Architecture Lead
**Primary Role:** Backend Core & Database Management
*   **Tech Stack:**
    *   **H2 Database**: Embedded, zero-install SQL database.
    *   **Spring Data JPA / Hibernate**: Object-Relational Mapping for database interactions.
    *   **Spring Security**: Authentication and authorization framework.
    *   **Docker Java API**: Library to control Docker containers programmatically.
*   **Usage:** Designing the schema, managing transactions, implementing security, and handling core infrastructure logic.

---

## 👤 Member 2: API Handling & Business Logic Developer
**Primary Role:** Backend Services & API Development
*   **Tech Stack:**
    *   **Spring Boot 3.x**: Framework for building standalone, production-grade Spring applications.
    *   **Spring Web MVC**: Model-View-Controller framework for building web applications.
    *   **Jackson**: JSON processor for Java.
    *   **Swagger / OpenAPI**: API documentation and design tools.
*   **Usage:** Creating REST controllers, implementing business logic (the "Brain"), rapid API development, and request validation.

---

## 👤 Member 3: Frontend Lead
**Primary Role:** Frontend UI/UX Design & Implementation
*   **Tech Stack:**
    *   **JavaFX**: Rich client application platform for desktop.
    *   **FXML**: XML-based language for defining user interfaces.
    *   **CSS**: Style sheets for customizing the look and feel.
    *   **Java HTTP Client**: Standard Java library for making HTTP requests.
*   **Usage:** Designing the UI, styling components and scenes, and connecting the frontend to backend APIs.
