MINICLOUD TEAM ROLES & TECH STACK (8-WEEK ACCELERATED)

MEMBER 1: DATABASE & CORE ARCHITECTURE LEAD
Primary Role: Backend Core, Infrastructure & Database Management
Tech Stack:
- H2 Database: Embedded SQL database for platform state.
- Spring Data JPA: Object-Relational Mapping.
- Spring Security: JWT Authentication and authorization.
- Docker Java API: Core engine to provision Compute (Tomcat) and Database (MySQL) containers.
Usage: Designing DB schemas, managing transactions, security, and interacting directly with the host's Docker daemon.

--------------------------------------------------

MEMBER 2: API HANDLING & BUSINESS LOGIC DEVELOPER
Primary Role: Backend Services, API Routing & Networking
Tech Stack:
- Spring Boot 3.x: Standalone Spring applications.
- Spring Web MVC: REST web applications.
- Spring Cloud Gateway / Java Proxy: Dynamic load balancing and HTTP traffic routing.
- Jackson & OpenAPI: JSON processing and API documentation.
Usage: Creating REST controllers, building the Load Balancer logic, validating requests, and enforcing quotas.

--------------------------------------------------

MEMBER 3: FRONTEND LEAD
Primary Role: Frontend UI/UX Design & Implementation
Tech Stack:
- JavaFX: Hardware-accelerated desktop client application platform.
- FXML & CSS: Defining modern user interfaces and styling (e.g., Dark Mode).
- Java HTTP Client: Connecting frontend actions to backend APIs.
Usage: Designing dashboards, real-time CPU/RAM charts, drag-and-drop storage zones, and packaging the final executable.
